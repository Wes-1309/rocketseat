Para dominar os conceitos fundamentais que citaremos a seguir é interessante ter um editor de código, um navegador e o nodejs instalados e configurados. Vamos aos fundamentos com alguns exemplos práticos. Estrutura de dados Vamos a modelagem de estrutura de dados com javascript. Escolhido o javascript pois trata-se de uma linguagem da web, é poderosa sendo de alto nivel. Permitindo ter foco no entendimento e não na construção. Cuidar da estrutura de dados é ter a preocupação com três etapas: modelar a estrutura, instanciar a estrutura e criar funcionalidades dessa estrutur, assim, os dados seram gerenciados. As estrutura mais utilizadas são: array: ["jupter", "marte"] matriz: [ ["2", "4"], ["8", "9"] ] stack(pilha) - LIFO: push(), pop(), peek() queue(fila) - FIFO: enqueue(), dequeue() E para cada estrutura de dados, temos um pacote de métodos para manipular essas informações. Além de ser muito importante as funções que criamos para realizar operações conforme as regras que estabelecemos para nossa aplicação. As estruturas no Código [Fazer] A estrutura de dados é importante para gerenciar os dados, as informações da nossa aplicação. Tarefas como, organizar, ordernar, buscar são algumas funcionalidades básicas que contribuem e muito para melhorar a escrita dos algoritmos, melhorar a eficiência dos nossos aplicativos.
